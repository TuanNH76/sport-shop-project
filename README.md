# sport-shop-project
# Online Sport Shop BackEnd
A Online Shop web application backend using Spring Boot.
# Feature
- REST API
- JWT authentication
- Cart & order management
- Checkout
- Catalogue
- Order management
- Pagination
- Excel Export
# Technology Stack
- Java 8
- Spring Boot 2.7
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- MySQL
- Maven
