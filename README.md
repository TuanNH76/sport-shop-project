# sport-shop-project
## Online Sport Shop
A Online Sport Shop Web Application.
## Feature
- REST API
- JWT Authentication
- Cart & Order management
- Checkout to Order
- Catalogue
- Pagination
- Excel Export
- 
## Technology Stack
### Backend
- Java 8
- Spring Boot 2.7
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL
- Maven
### Frontend
- TypeScript
- AngularJs
